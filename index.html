Fighter Jet Takes the Sky

This was the first project I actually coded for. This was made at hackBCA. My whole team was new to coding so we wanted to make a simple game. We found source code (we were hopeless without it) of the "Helicopter Game" and worked to make that the game we envisioned. Instead of a helicopter, we made a game where the user control a fighter jet and has to dodge missiles that gradually become faster. I was very proud of my team because we spent the first few hours of the hackathon clueless!

I've been working on the game on my own time at home. Here's the latest version! It's fast paced and addicting!

Elima-Click

My team made this game at HackRU. It's very similar to Tetris, but it doesn't use any keyboard keys. Instead a player clicks blocks one at a time to change their color by cycling through five certain colors. The player must get a row to be the same color to eliminate it. Each time the player clicks, a randomly colored block will drop from a random column. The goal is to get as high of a score as possible without the blocks reaching the top of the grid. This was made purely in Java.
