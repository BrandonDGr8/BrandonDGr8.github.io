Fighter Jet Takes the Sky

This was the first project I actually coded for. This was made at hackBCA 2014. My whole team was new to coding so we wanted to make a simple game. I've been working on the game on my own time at home. Here's the latest version! It's fast paced and addicting!

Elima-Click

My team made this game at HackRU Spring 2014. It's very similar to Tetris, but it doesn't use any keyboard keys. Instead a player clicks blocks one at a time to change their color by cycling through five certain colors. The player must get a row to be the same color to eliminate it. Each time the player clicks, a randomly colored block will drop from a random column. The goal is to get as high of a score as possible without the blocks reaching the top of the grid. This was made purely in Java.

You may download it here:

Download Elima-Click

Mathematical Formula Saver

This app is meant to do what the title says. It saves custom mathematical formulas so one can easily plug in numbers and variables and solve an equation. It is supposed to save one's time so they do not have to type a formula over and over again on a calculator. They can simply save the formula once, then plug in only the variable numbers to quickly solve equations. This app was built in Java at CodeDayNY Spring 2014.
